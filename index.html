How to embed

Use ?target=HH:MM:SS to set a fixed UTC deadline. All viewers count down to the same moment regardless of timezone — ideal for shared embeds.

Use ?end=30m (or 1h, 1h30m) to set a deadline relative to when the page first loads. This is ideal for Notion templates — embed it once and every copy of the template will start its own fresh countdown when opened, with no manual URL editing needed. The timer is locked in on first load, so refreshing won't reset it.

After expiry the clock counts negative and flashes. Additional passive aggressive measures will be added until meetings end on time.
